Vokabeltrainer

Ein Vokabeltrainer in C++ mit Qt, entwickelt im Rahmen von Programmieren 2 zur Anwendung fortgeschrittener C++ Konzepte.

Features
- Grafische Benutzeroberfläche mit Qt
- Verwaltung und Abfrage von Vokabeln

Verwendete Konzepte 
- Mehrere Klassen mit klarer Verantwortungsteilung
- Operatorüberladung
- Rule of Five (Konstruktoren, Destruktor, Copy-/Move-Semantik)
- Templates (generische Programmierung)
- Eigene Iteratoren für Container-Klassen
- Implementierung von Algorithmen und Datenstrukturen
- etc.
